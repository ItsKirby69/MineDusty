# Contributing 

To Get Started:
1. Fork this repository on the **top-right** at the main page.
2. Do your intended changes or additions.
3. Create a pull request:
   - Bring **showcase** and **descriptions** on changes
   - I will likely read your request within the day, so be patient.

## Contributing Rules
**Commit Messages** should reflect the present changes. I can forgive loose messages on extremely minor changes.

**Smaller Changes** are more likely to get accepted. Keep size of the PR small when possible.

**Significant** gameplay changes must be discussed before hand in the [forums](https://discord.com/channels/391020510269669376/1395862844427210862).

**Textures** must follow the exising styling (that is as vanilla looking as possible. Please avoid chimera sprites which involve merging vanilla sprites together)

**Bug fixes** are always welcome, as long as it fixes **unintended** behaviour.

Please follow basic formatting guidelines based on the existing code:
- Wildcards for imports `import this.package.*` when possible.
- Same-line braces (similar to Mindustry's formatting).
- Reduced whitespacing between methods.
- Tab (4 spaces) indentation.

## Notice
Contributions are not guaranteed to be accepted. 
Please understand that this mod is a passion project of mine which may limit major changes.
